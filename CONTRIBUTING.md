# Contributing to ContextFlow

We welcome contributions from the community! To ensure a smooth process, please follow these guidelines.

## How to Contribute
1. **Report Bugs**: Use GitHub Issues to report bugs or suggest enhancements.
2. **Submit Pull Requests**: 
   - Fork the repository.
   - Create a feature branch (`git checkout -b feature/amazing-feature`).
   - Commit your changes (`git commit -m 'Add amazing feature'`).
   - Push to the branch (`git push origin feature/amazing-feature`).
   - Open a Pull Request.

## Development Setup
1. Install Rust (latest stable version).
2. Clone the repository.
3. Build the project: `cargo build --release`.
4. Run tests: `cargo test`.

## Code Style
- Follow standard Rust formatting (`cargo fmt`).
- Ensure all code is documented with clear comments.
- Maintain a technical, objective tone in all documentation.

## Community & Conduct
Please be respectful and professional in all interactions. 

Thank you for helping improve ContextFlow!
